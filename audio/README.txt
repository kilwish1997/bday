HOW TO ADD YOUR LOVE SONG
=========================

1. Copy your audio file (song) into this folder

2. Rename it to: love-song.mp3
   (or love-song.ogg for OGG format)

3. Supported formats:
   - MP3 (.mp3) - Most common
   - OGG (.ogg)
   - WAV (.wav)
   - M4A (.m4a)

4. The song will play automatically when you click the "Play Love Song" button

5. The song will loop continuously (repeat)

6. Click again to pause

NOTE: Make sure the file is named exactly "love-song.mp3" for it to work!
