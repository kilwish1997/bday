## Setup
1. Add your photos to the `images/` folder (photo1.jpg through photo12.jpg)
2. Add your videos to the `videos/` folder (video1.mp4, video2.mp4)
3. Add your love song to the `audio/` folder (love-song.mp3)
4. Open `birthday-love.html` in a browser

## Customization
Edit the JavaScript section in `birthday-love.html`:
- `BIRTHDAY_DATE` - Her birthday date
- `GIRLFRIEND_NAME` - Her name
- `photoDescriptions` - Captions for each photo
- `wishes` - Customize the 15 wish messages
